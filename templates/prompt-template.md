# プロンプト: <短いタイトル>

- **目的:** このプロンプトで何を達成するか（1〜2 行）
- **想定モデル:** 例）GPT-5 / Claude Opus 4.7 / Gemini 3 Pro
- **教育学的根拠:** どの学習理論・設計モデルに立脚するか
- **想定ユーザー:** 教員 / 教材開発者 / 学習者 …
- **ライセンス:** [CC BY-NC 4.0](../../../LICENSE)

## 使い方

1. `<変数>` の部分を置換する
2. モデルに与える前に、対象学年・教科・時間などの前提を明示する
3. 出力後は評価ルーブリック（末尾）で検証する

## プロンプト本体

```text
あなたは <役割> です。
以下の条件で <タスク> を行ってください。

# 前提
- 対象学習者: <学年・レベル>
- 教科・単元: <教科・単元>
- 到達目標 (Learning Outcome): <観測可能な行動で記述>

# 教育学的制約
- ブルームのタキソノミー <段階> を対象とする
- 形成的評価を <頻度・方法> で組み込む

# 出力形式
- 見出し付き Markdown
- 各セクション末尾に「教員向けメモ」を付ける
```

## 期待される出力の要件

- [ ] 到達目標と整合している
- [ ] 認知的負荷が学年に対して適切
- [ ] 評価方法が観測可能
