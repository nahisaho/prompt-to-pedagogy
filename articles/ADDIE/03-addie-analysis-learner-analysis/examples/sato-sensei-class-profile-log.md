## 1. 入力: 匿名化されたクラスタメモ (架空)

### 1.1 学級の基礎情報

| 項目 | 値 |
| :-- | :-- |
| 校種・学年 | 高校 2 年 |
| 教科・コマ数 | 国語 (現代文・古典), 週 4 コマ |
| 学級規模 | 40 名 (男子 21 / 女子 19) |
| 到達度分布 (前学年末) | 上位 25% / 中位 50% / 下位 25% |
| 教員が伸ばしたい領域 | 批判的思考の言語化 |
| 次に扱う単元 | 「羅生門」(芥川龍之介), 8 コマ |

### 1.2 クラスタ (7 群)

```
C1 (n=8) 読解精度高・発言慎重群
  特徴: 記述の論理構造が明確、本文根拠を探す習慣あり
  課題: 発表・討論で自分の解釈を主張する経験が少ない

C2 (n=10) 発言活発・根拠曖昧群
  特徴: グループワークで積極的、雰囲気作りが上手
  課題: 「なぜそう読めるか」を本文に戻って示すこと

C3 (n=6) 興味の波が大きい群
  特徴: 関心が重なる場面で深い読みを見せる
  課題: 教材と自分をつなぐ問いの提示

C4 (n=5) 標準的・低変動群
  特徴: 課題を淡々とこなす、目立たない
  課題: 自分の意見を持つ経験が少ない

C5 (n=4) 古典志向・現代文苦手群
  特徴: 古典の記憶事項に強い、現代文の解釈で戸惑う
  課題: 現代文での "解釈は複数ありうる" 感覚

C6 (n=4) 語彙負荷高群
  特徴: 語彙が制限的、読み速度が遅い
  課題: 語注支援・音読機会の確保

C7 (n=3) 表現多様性志向群
  特徴: 記述より口頭・図で表現するほうが得意
  課題: 記述評価中心の授業構造で埋もれがち
```

## 2. プロンプト 01 の実行 (学級プロファイル合成)

### 2.1 モデルの応答 (Claude Opus 4.7)

#### 学級全体の傾向 (5 行以内)

2 年 A 組は、**「読める群と話せる群が分かれている」** 傾向が示唆される。
上位群 (C1) は精緻な読解ができるが自己主張が弱く、
中位群 (C2) は発言力があるが根拠提示が弱い。両群を **相互に依存させる設計**
をすると、双方が伸びる可能性が高い。下位群には興味の波が大きいクラスタ (C3) と、
語彙・読み速度に課題を持つ群 (C6, C7) が混在しており、
一律の「遅れフォロー」では届きにくい可能性がある。

#### 教師が押さえるべき 5 つのポイント

| # | 学級傾向 | 授業設計への示唆 | 根拠 (クラスタ) |
| - | :-- | :-- | :-- |
| 1 | 読解群と発言群が分離している傾向 | ペア構成で意図的に混ぜる。互恵的依存を作る | C1 × C2 |
| 2 | 発言活発群の根拠提示が弱い傾向 | 「本文の何行目？」を発問の定型に組み込む | C2 |
| 3 | 興味依存で集中が変動する層がある | 教材と "自分" をつなぐ導入発問を必須化 | C3 |
| 4 | 語彙・読速に負荷を感じる層がある | 語注の事前提示、音読機会の確保 | C6, C7 |
| 5 | 上位群のアウトプット機会が不足 | ジグソー法で C1 に説明役の場を作る | C1 |

#### クラスタ間の相互依存の可能性

- **C1 × C2:** C1 が根拠提示の型を示し、C2 が発表の勢いを提供 / 摩擦: C1 が消極的になる懸念
- **C3 × C5:** C3 の関心接続力が C5 の現代文への戸惑いを緩和 / 摩擦: 話題が私的になりすぎる懸念
- **C7 × C6:** C7 の多様な表現手段が C6 の語彙負荷の代替経路になる / 摩擦: 評価公平性の議論

#### このデータ範囲では判断しないほうがよい論点

- 個々の生徒の家庭背景・診断情報 (メモに含まれないため推測を差し控える)
- 特定生徒の指導方針 (プロファイル作成の目的外)
- 生徒同士の人間関係 (観察メモ範囲では把握できない)

#### メタコメント

- 現在のメモは "授業内観察" が中心。**授業外での学習ログ (家庭学習時間・提出物完了率)**
  を追加すると、C4 (低変動群) の実像が具体化する可能性がある
- クラスタ n=3〜4 の群 (C5, C6, C7) は個人差が大きくなるため、
  集約傾向としては仮説の確度が低いことを念頭に運用してください

## 3. 反証プロンプトの実行

### 3.1 モデルの応答 (Claude Opus 4.7)

**5 つのポイントの各行への反例:**

| # | 主張 | 誤っている可能性 |
| - | :-- | :-- |
| 1 | ペアで混ぜると相互に伸びる | C1 が C2 の発言力に圧倒されて発話量がさらに減少する可能性 |
| 2 | 「本文の何行目？」を発問に組み込む | C2 が発言量そのものを減らしてしまう可能性 (安心して発言できる場が縮む) |
| 3 | 導入で "自分" と教材をつなぐ | C3 以外の生徒には冗長に感じられ、集中が切れる可能性 |
| 4 | 語注の事前提示 | C6 の "分からない語を自分で調べる" 学習機会が失われる可能性 |
| 5 | ジグソー法で C1 に説明役 | C1 が「教える技術」を持たないまま説明役に立たされる負担 |

**クラスタ分けそのものへの疑い (2 件):**

- 40 人を 7 クラスタに畳むと、各クラスタ内の分散 (特に C1 内の "静かだが批判的" と
  "静かで無関心" の違い) が失われる
- クラスタは "観察された傾向" であり、**「そのクラスタに入りきらない生徒」** が
  存在する。特にクラスタ境界の生徒 (例: C2 と C3 の間) は、時期によって帰属先が変わる

**根拠が薄い推測項目 (AI の自己申告):**

- C7 の「表現多様性志向」— n=3 のサンプルで一般化するのは統計的に脆弱
- C4 の「低変動群」— "目立たない" ことは "変動が小さい" こととは限らない

**4 週間後のプロファイル更新手順の提案:**

1. 単元「羅生門」の中で、クラスタ間の境界にいる生徒 (推定 5〜7 名) を観察対象に設定
2. Week 2 と Week 4 に、クラスタ帰属の見直しメモを追加 (原本ではなく傾向カテゴリで)
3. プロンプト 01 を **同じ入力形式** で再実行し、AI の解釈が変わるかを確認
4. AI 出力ではなく "自分の観察" のほうを優先し、プロファイルを更新
5. 更新履歴を残す (何を根拠に何を変えたか)

## 4. プロンプト 02 の実行 (ZPD 仮説生成)

### 4.1 モデルの応答 (Claude Opus 4.7)

#### 単元全体の ZPD マップ

「羅生門」単元における学級の "次の一歩" は、**"複数の解釈が並立しうる"**
というテクスト理解と、それを **本文根拠でどう支えるか** の言語化と想定される。
C1 群は他者の解釈への応答、C2 群は根拠提示、C3・C5 群は解釈の複数性そのものへの気づき、
C6・C7 群は表現経路の確保が焦点になる可能性がある。

#### クラスタごとの ZPD 仮説 (抜粋)

| クラスタ | 一人でできること | ZPD (支援があればできる) | 支援策 (教師) | 支援策 (他生徒) |
| :-- | :-- | :-- | :-- | :-- |
| C1 | 本文根拠を挙げて自分の解釈を書く | 他者の解釈を批判的に検討し、自分の解釈を **並列に置く** | ミニ討論の型を提示 | C2 とのペアで反対意見を受ける |
| C2 | 直感的な解釈を口頭で語る | 自分の解釈を本文の複数箇所で支える | 「根拠 3 点セット」の問いかけ | C1 の記述を見て "型" を学ぶ |
| C3 | 関心に触れれば深い読み | 興味の外にも自分の問いを立てる | 導入 3 分の "自分ゴト化" ワーク | C5 と組んで話題を広げる |
| C4 | 課題を淡々とこなす | 自分の解釈を "持つ" ことに意識を向ける | 選択式の解釈提示 → 記述への段階化 | C2 の発話に触れる場 |
| C5 | 古典的な形式に強い | 現代文で「解釈は複数」を体験する | 対比的解釈を教師が並列提示 | C3 との対話 |
| C6 | 支援があれば理解可能 | 自分で語彙を推測する経験を持つ | 語注を "後出し" 化 | C7 の非言語表現を経由 |
| C7 | 図・口頭で表現できる | 表現を "本文根拠と結ぶ" 経験を持つ | ビジュアル → 記述への橋渡し課題 | C1 との共同表現 |

#### Fading (足場を外していく) 段階

- **コマ 1-2:** 濃い足場 — 教師が解釈を 3 つ並列提示、根拠位置も指示
- **コマ 3-5:** 部分的な足場 — 解釈候補は生徒が挙げ、根拠は教師が問い返す
- **コマ 6-8:** 足場を外す — 相互評価で解釈と根拠を検討、教師は観察者へ

#### メタコメント

- C1 内の分散 (静かな批判者 vs. 静かな無関心) は Fading 中に露わになる可能性が高い
- 単元 Week 2 (コマ 3-4) で ZPD 仮説の修正を推奨。特に C3 と C5 の帰属

### 4.2 反証出力 (抜粋)

- **C1 の ZPD が過大の可能性:** 「批判的検討」は 2 年生の発達段階では難易度が高く、
  **「異なる解釈を並列で扱える」** 段階が妥当かもしれない
- **C4 の ZPD が過小の可能性:** "自分の解釈を持つ" は既にできている可能性があり、
  観察が不十分なまま低く見積もっている
- **形式主義に陥りやすい支援策:** C2 の「根拠 3 点セット」は、
  数を求めることで引用の羅列に流れる懸念
- **修正が必要になるサイン (3 件):**
  1. コマ 3 までに C1 の発話量が変化しない → ペア構成の見直し
  2. C2 の記述に "本文引用" が形式的にコピペされる → 問いの立て方を変える
  3. C3 の集中の波が単元全体を貫いてしまう → 導入の "自分ゴト化" だけでは足りない

## 5. プロンプト 03 の実行 (UDL 多様性チェック)

### 5.1 モデルの応答 (Claude Opus 4.7)

#### Engagement の穴

- **穴:** 関心の入り口が「教材テーマ (羅生門)」1 本に絞られている。
  C3 以外にも意欲点火に別ルートが必要な生徒がいる可能性 (影響: C3, C4, 一部の C5)
- **穴:** 「選ぶ経験」が学級全体で少ない (発問・活動・評価すべて教師指定) (影響: 学級全体)

#### Representation の穴

- **穴:** テキスト中心で、視覚・聴覚・実物を経由する提示がない (影響: C6, C7)
- **穴:** 語彙負荷の高い生徒への **事前補助が明示されていない** — 現在は "後出し語注"
  の運用だが、フォールバック手段がない (影響: C6)

#### Action & Expression の穴

- **穴:** 評価が **記述一択**。口頭・ビジュアル・演劇的表現が不在 (影響: C7)
- **穴:** 上位群 (C1) の "説明する" 表現機会は増えたが、他クラスタが
  多様に表現するルートは未整備 (影響: C2, C3, C7)

#### 実装可能な調整案

| # | 調整案 | 対象 | 効果 | 教師の負担 |
| - | :-- | :-- | :-- | :-- |
| 1 | 単元の最終アウトプットを「記述 or 図解」の選択式にする | C7 中心・全員影響 | 表現多様性の担保、C7 の埋没回避 | 中 (評価基準を 2 種類作る必要) |
| 2 | 導入 3 分の "自分ゴト化ワーク" に **選択肢 3 つ** を用意 | C3 中心・全員影響 | 意欲点火のルート増加、選択経験 | 低 (準備は既存) |
| 3 | 語注の "後出し" と "事前提示" を、生徒が選べるようにする | C6 中心 | 語彙負荷の緩和、自律性の担保 | 低 (資料は同じ) |

#### 調整によって新たに生まれるリスク

- **リスク A:** 選択肢が増えることで、C4 (低変動群) が "選ぶこと自体" に負担を感じる可能性
  → 対処: 迷ったときのデフォルトを教師が用意
- **リスク B:** 評価が 2 種類 (記述・図解) になることで、公平性への疑問が生徒側から出る可能性
  → 対処: ルーブリックの共通観点を先に共有 (第7回の内容と接続)

#### メタコメント

- 「全員に全選択肢」を避けるための基準は、**"必要な生徒に必要な調整"**。
  今回は C6 と C7 が明らかに埋没していることが判定基準
- UDL は "学習の場の設計" を扱うが、生徒間の人間関係や家庭学習環境は扱えない。
  それらは第5回 (環境・制約分析) の範疇

### 5.2 反証出力 (抜粋)

- **取りこぼしの過剰指摘:** 「記述一択が問題」は国語の学問的伝統からは自然で、
  過度な多様化は教科の本質を薄める可能性がある
- **新たな不公平:** 図解表現を選んだ生徒に「記述評価では損」という感覚を与える恐れ
- **UDL が扱えない論点:** 学級運営の根本 (発言のしやすさ・心理的安全性) は
  UDL の枠組み外
- **「まず 1 つだけなら」:** 調整案 #2 (導入ワークの選択肢 3 つ) を推奨。
  教師の負担が最小で、学級全体に効果が及ぶ

## 6. 3 プロンプトの結果を統合したメモ (佐藤先生)

> - C1 × C2 の互恵ペアリングを軸に置く
> - 「本文根拠」の要求は "型" ではなく "問いの言い換え" で運用
> - UDL 調整は #2 (導入選択肢) から始める
> - Week 2 でクラスタ帰属を見直し、Week 4 でプロファイル更新
> - AI 出力は "たたき台"。実際の生徒観察を最優先
