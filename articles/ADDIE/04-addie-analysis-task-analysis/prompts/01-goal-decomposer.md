# プロンプト 01: 目標分解 (Goal Decomposer)

- **目的:** 抽象的な学習目標を、Bloom's Taxonomy 改訂版に基づく **観察可能な動詞** に分解する
- **想定モデル:** GPT-5 / Claude Opus 4.7 / Gemini 3 Pro
- **教育学的根拠:**
  - Bloom's Taxonomy 改訂版 [[Anderson & Krathwohl 2001]](../../../shared/references.md#anderson-krathwohl-2001)
  - Mager の目標記述法 [[Mager 1997]](../../../shared/references.md#mager-1997)
  - ADDIE モデルの Analysis フェーズ
- **ライセンス:** [CC BY-NC 4.0](../../../LICENSE)

---

## 使い方

1. 手元の指導計画から、**分解したい目標** を 1 つ選ぶ
2. 単元名・コマ数・教科・学年など、公開可能な文脈情報を用意
3. 下のテンプレートの `<...>` を置換して LLM に投入
4. 反証プロンプトを続けて実行
5. 出力の動詞候補から、教師自身が **6〜8 個に絞り込む**

> [!IMPORTANT]
> 生徒個人の情報は投入不要です。単元目標・学年・教科・教材は原則として
> **公開情報** (指導計画・教科書会社の資料) の範囲で足ります。

---

## プロンプト本体

````text
あなたは、Bloom's Taxonomy 改訂版とインストラクショナルデザインに精通した
教員研修の専門家です。

# タスク
以下の学習目標を、Bloom's Taxonomy の 6 段階 (記憶/理解/応用/分析/評価/創造) に
分解し、各段階について **観察可能な動詞** で書かれた具体的目標を提案してください。

# 前提と制約 (厳守)
- 動詞は「観察できる行動」を指すもの (要約できる/比較できる/反論できる 等)
- 「理解する」「感じる」「気づく」など観察不可能な動詞は使わない
- 6 段階すべてを埋める必要はない。単元にとって不自然な段階は「該当なし」で良い
- 単元コマ数に対して過剰な分解を避ける (目安: 6〜8 個の動詞)
- 各動詞に「評価タイミング (どのコマで確認するか)」を添える
- 目標に含まれる **暗黙の前提** を最初に明文化する
- **態度目標** (愛する・尊重する等) は Bloom で扱いにくいため、
  「単元目標としての観察可能性」と「授業で扱う背景コンセプト」を分けて記述
- 語尾は非断定 ("〜が想定される" "〜と解釈できる")
- 出力は日本語

# 入力: 学習目標 (現状のふわっと版)
<例: 「批判的思考を伸ばし、多面的な読みができるようにする」>

# 入力: 単元・文脈情報
- 教科・学年: <例: 高校 2 年 現代文>
- 単元名: <例: 「羅生門」(芥川龍之介)>
- コマ数: <例: 8 コマ>
- 教材の位置づけ: <例: 主教材、二学期>
- 学級の粗い傾向 (匿名化, 任意): <例: 読解群と発言群が分かれる>

# 出力形式

## 1. 暗黙の前提の明文化 (3-5 項目)
目標に含まれる暗黙の意味を箇条書きで整理

## 2. Bloom 段階別の観察可能な動詞

| Bloom | 単元での動詞化 (目標) | 評価タイミング (コマ) |
| :-- | :-- | :-- |
| 1. 記憶  |    |    |
| 2. 理解  |    |    |
| 3. 応用  |    |    |
| 4. 分析  |    |    |
| 5. 評価  |    |    |
| 6. 創造  |    |    |

## 3. この分解で扱えない目標 (2-3 件)
- 態度目標や協働目標など、Bloom では扱いにくいもの
- 「削除する」か「授業の背景コンセプトとして残す」の判断助言

## 4. メタコメント (2-3 行)
- 分解が過剰でないか、単元の全体像を保っているか
````

## 単発モード (1 目標だけ試す)

指導計画の目標を 1 つだけ試す簡易版:

````text
以下の学習目標について、Bloom の 6 段階のうち **もっとも該当しそうな 2〜3 段階**
に絞り、観察可能な動詞を各段階 1 つずつ提案してください。
その動詞が「観察可能」と言える根拠も添えてください。

目標: <ここに 1 目標>
教科・学年: <>
````

---

## 追加プロンプト: 反証

```text
先ほどの目標分解について、以下を実行してください。

1. 各動詞が実際には観察困難である可能性を 1 つずつ挙げる
2. Bloom の "上位" (評価・創造) に偏っている可能性を検証
3. 提案動詞が単元の "全体像" を失わせている可能性を挙げる
4. 分解された動詞を "縮約テスト" (1 文で単元の学びを説明) にかけたときの
   予測される要約を書く

語尾は決定論的な断定を避けること。
```

---

## 期待される出力の要件

- [ ] 動詞がすべて「観察可能」な形になっている
- [ ] 各動詞に評価タイミングが明記されている
- [ ] 「扱えない目標」の項に態度目標への言及がある
- [ ] 反証出力の "縮約テスト" が単元の魂を捉えている

## 評価ルーブリック (簡易)

| 観点 | 3: 優 | 2: 良 | 1: 要改善 |
| :--- | :---- | :---- | :-------- |
| 動詞の観察可能性 | すべて明確 | 過半数明確 | 曖昧多数 |
| Bloom の網羅性 | 単元に必要な段階を選択 | 一部偏り | 上位/下位に極端 |
| 態度目標の扱い | 別立てを明示 | 混在 | 混同 |
| 縮約可能性 | 1 文で戻せる | 部分的 | 戻せない |
