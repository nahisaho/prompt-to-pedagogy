# プロンプト 03: 「観察可能か」を厳しく問う (Observable Behavior Check)

- **目的:** 分解された各目標について、**本当に観察可能か・評価可能か** を厳格に検証し、曖昧な動詞や態度目標を明らかにする
- **想定モデル:** GPT-5 / Claude Opus 4.7 / Gemini 3 Pro
- **教育学的根拠:**
  - Mager の ABCD 目標記述法 [[Mager 1997]](../../../shared/references.md#mager-1997)
  - Bloom 改訂版の "動詞分類" [[Anderson & Krathwohl 2001]](../../../shared/references.md#anderson-krathwohl-2001)
  - Gagné の 5 分類 (態度領域を分離する根拠) [[Gagné 1985]](../../../shared/references.md#gagne-1985)
- **ライセンス:** [CC BY-NC 4.0](../../../LICENSE)

---

## 使い方

1. プロンプト 01 の出力 (動詞化された目標リスト) を用意
2. 下のテンプレートに貼り付けて LLM に投入
3. 反証プロンプトを続けて実行
4. ⚠️ や ❌ が付いた目標を、教師自身で修正 or 別立てにする

> [!IMPORTANT]
> このプロンプトは **意図的に厳しい態度で判定** させます。
> 「これも観察できる」と甘く言われるより、
> 「ここは曖昧」と厳しく言われるほうが、単元設計は堅牢になります。

---

## プロンプト本体

````text
あなたは、教育評価とインストラクショナルデザインに精通した厳格な査読者です。

# タスク
以下の動詞化された目標リストの一つひとつについて、
**「その目標は本当に観察可能か・評価可能か」** を厳しく判定してください。
甘い判定は避け、曖昧さを明らかにすることを優先してください。

# 判定基準
- ✅ 観察可能: 評価者間で判定が一致する明確な行動基準がある
- ⚠️ 部分的: 行動は見えるが、質を測る基準が未定義 (追加基準で救える)
- ❌ 観察不可: 態度・価値観・内的状態を指しており、この形では評価不能

# 前提と制約 (厳守)
- 各目標について、上記 3 段階で判定
- ⚠️ と ❌ には、具体的にどこが曖昧か・何が観察できないかを明記
- ⚠️ は「追加基準を加えれば ✅ になる」道筋を示す
- ❌ は「削除する」ではなく「別領域 (態度・背景コンセプト等) に移す」を提案
- 「観察不可 = 悪い」ではなく、「単元目標として評価される形かどうか」の判定
  であることを明示
- 語尾は非断定 (判定自体は厳格に、根拠は "〜と解釈できる")
- 出力は日本語

# 入力: 動詞化された目標リスト (プロンプト 01 の出力)
<動詞リストを貼り付け>

# 出力形式

## 判定表

| # | 目標 | 判定 | AI からの指摘 |
| - | :-- | :-: | :-- |
| 1 |    |    |    |

## ⚠️ 目標への追加基準の提案

各 ⚠️ 目標に対して、「これを加えると ✅ になる」基準を具体的に提案:
- 目標 #x: 追加基準 = ...

## ❌ 目標の扱いの提案

各 ❌ 目標に対して、どこに位置づけるべきか助言:
- 目標 #x: 単元目標から外し、xxx として保持

## 全体助言 (2-3 行)
- 目標リスト全体としての "観察可能性" のバランス
- 教師が判定を上書きすべき場面
````

---

## 追加プロンプト: 反証

```text
先ほどの観察可能性の判定について、以下を実行してください。

1. ✅ 判定の中に、実は観察が難しいものが混じっていないか再検証
2. ❌ 判定を厳しくしすぎて、教育的に重要なものを排除している可能性を検討
3. 「観察可能」だけを優先すると、単元が痩せる懸念がないか意見を述べる
4. 教師がこの判定結果を上書きすべき場面を 3 つ挙げる

語尾は決定論的な断定を避けること。
```

---

## 期待される出力の要件

- [ ] すべての目標に判定 (✅/⚠️/❌) が付いている
- [ ] ⚠️ には具体的な追加基準の提案がある
- [ ] ❌ には「削除ではなく別立て」の助言がある
- [ ] 反証で「観察可能主義の副作用」への言及がある

## 評価ルーブリック (簡易)

| 観点 | 3: 優 | 2: 良 | 1: 要改善 |
| :--- | :---- | :---- | :-------- |
| 判定の厳格さ | 曖昧を漏らさず指摘 | 部分的 | 甘い判定 |
| 追加基準の具体性 | ⚠️ が救済可能 | 抽象的 | 一般論 |
| ❌ の扱い | 別立てを提案 | 削除提案 | 混乱 |
| 反証の質 | 副作用まで言及 | 部分的 | 追従的 |
