# プロンプト 02: 前提知識・下位スキルの依存グラフ (Prerequisite Mapper)

- **目的:** 動詞化された目標に対して、**「これができるためには何ができている必要があるか」** を再帰的に問い、下位スキルと前提知識の依存関係を可視化する
- **想定モデル:** GPT-5 / Claude Opus 4.7 / Gemini 3 Pro
- **教育学的根拠:**
  - Gagné の学習階層 (Learning Hierarchy) [[Gagné 1985]](../../../shared/references.md#gagne-1985)
  - Bloom 改訂版 [[Anderson & Krathwohl 2001]](../../../shared/references.md#anderson-krathwohl-2001)
  - Vygotsky の ZPD (足場の設計) [[Vygotsky 1978]](../../../shared/references.md#vygotsky-1978)
- **ライセンス:** [CC BY-NC 4.0](../../../LICENSE)

---

## 使い方

1. プロンプト 01 の出力 (Bloom 分解済み動詞リスト) を用意
2. 学級プロファイル (第3回のクラスタ) を用意 (匿名化済み)
3. 下のテンプレートに貼り付けて LLM に投入
4. 反証プロンプトを続けて実行
5. 出力の依存グラフから、単元前半で先に扱う「前提」を特定

> [!TIP]
> 依存グラフは **仮説** です。実際の生徒の詰まりどころは授業で観察するまで
> 確定しません。「単元前半で先に確認する候補」として使い、Week 1 で修正
> することを前提に運用してください。

---

## プロンプト本体

````text
あなたは、学習科学と Gagné の学習階層に精通した授業設計の専門家です。

# タスク
以下の動詞化された目標に対して、**「これができるためには何ができている必要が
あるか」** を再帰的に問い、下位スキルと前提知識の依存関係を階層構造で
可視化してください。

# 前提と制約 (厳守)
- 分解は最大 3 階層まで (それ以上は過剰分解)
- 「前提」には「下位スキル」と「前提知識」の 2 種類があることを区別する
- 前提と目標の間に固定的な順序があると決めつけない (往還可能性を残す)
- 学級プロファイルのクラスタと接続し、**どのクラスタがどの前提でつまずきやすいか** を仮説として示す
- クラスタとの接続は「示唆」であり、クラスタ内個人差を考慮する
- 語尾は非断定
- 出力は日本語

# 入力: プロンプト 01 の出力 (Bloom 分解された動詞リスト)
<Bloom 段階別動詞リストを貼り付け>

# 入力: 学級プロファイル (第3回で作成した匿名化クラスタ)
<クラスタ C1〜C7 の傾向を貼り付け>

# 入力: 単元コマ数
<例: 8 コマ>

# 出力形式

## 依存グラフ (テキスト表現)
```
【目標: xxx】
  ├── 前提スキル/知識: yyy
  │     ├── さらなる前提: zzz
  │     └── ...
  └── ...
```
(目標ごとに 1 つ、最大 3 階層)

## つまずきの候補マップ

| つまずき位置 | 影響を受けやすいクラスタ | 対策の候補 | 対応コマ (推奨) |
| :-- | :-- | :-- | :-- |
|    |    |    |    |

## 「先に扱うべき前提」の優先順位 (Top 3)
1. 前提名 / 理由 / 学級全体への影響度

## メタコメント
- 依存グラフの限界 (順序の固定化リスク、クラスタ内個人差)
- Week 1 で修正するためのモニタリング指標
````

---

## 追加プロンプト: 反証

```text
先ほどの依存グラフについて、以下を実行してください。

1. 各前提が「実は前提として不要」である可能性を 1 つずつ挙げる
2. 依存の順序が固定的に見える箇所を指摘し、往還可能性を示す
3. クラスタとつまずきのマッピングが過度の一般化になっている可能性を検討
4. 「単元を進めながら前提が身につく」領域があれば列挙する

語尾は決定論的な断定を避けること。
```

---

## 期待される出力の要件

- [ ] 依存グラフが 3 階層以内に収まっている
- [ ] つまずきマップがクラスタと接続されている
- [ ] 優先順位 Top 3 が明示されている
- [ ] 反証出力で「不要な前提」への言及がある

## 評価ルーブリック (簡易)

| 観点 | 3: 優 | 2: 良 | 1: 要改善 |
| :--- | :---- | :---- | :-------- |
| 依存関係の妥当性 | 教科の論理と整合 | 部分的整合 | 乖離 |
| 過剰分解回避 | 3 階層以内 | 一部深すぎ | 深すぎ多数 |
| クラスタ接続 | 具体的で仮説と明示 | 抽象的 | 断定的 |
| 実務適用性 | Week 1 の行動が見える | 抽象的 | 一般論のみ |
