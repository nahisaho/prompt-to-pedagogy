# 用語集 (Glossary)

プロンプト設計と教育学の両分野で用いられる主要用語の対応表。
新しい記事で用語を導入する際は、まずここに追記してから本文で参照する。

## 教育学の用語

| 用語 | 英語 | 概要 |
| :--- | :--- | :--- |
| ブルームのタキソノミー | Bloom's Taxonomy | 認知領域の学習目標を 6 段階で分類 |
| ADDIE モデル | ADDIE Model | 分析・設計・開発・実施・評価のインストラクショナルデザイン枠組み |
| 形成的評価 | Formative Assessment | 学習の途中で行い、指導改善に用いる評価 |
| 総括的評価 | Summative Assessment | 学習の終末で到達度を判定する評価 |
| 認知的徒弟制 | Cognitive Apprenticeship | 熟達者の思考過程を可視化して学ぶ枠組み |
| 足場かけ | Scaffolding | 学習者の発達水準に応じた支援 |
| 到達目標 | Learning Outcome | 観測可能な行動として記述された学習成果 |

## プロンプト設計の用語

| 用語 | 英語 | 概要 |
| :--- | :--- | :--- |
| ロール指定 | Role Prompting | モデルに役割を与える技法 |
| Few-shot | Few-shot Prompting | 少数の例示を含めて出力形式を制御 |
| Chain-of-Thought | CoT | 思考過程を段階的に出力させる |
| 出力スキーマ | Output Schema | JSON 等の構造化出力を強制する制約 |
| システムプロンプト | System Prompt | 会話全体を規定する上位指示 |

## 対応関係の候補（記事で深掘りするテーマ）

- ロール指定 ⇔ 認知的徒弟制における「モデリング」
- Few-shot ⇔ 例示による概念形成 (Concept Attainment)
- 出力スキーマ ⇔ ルーブリック評価
- CoT ⇔ メタ認知の外化
